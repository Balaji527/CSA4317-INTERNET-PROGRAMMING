This code includes tools and examples of experiments related to the bird image dataset in a wind farm.
Dataset's url: http://bird.nae-lab.org/dataset

This code can run on Visual Studio and Windows machine.

contents:
WindEnergy2016Codes.sln -- VS solution for running evaluation code for AdaBoost
CaffeModels -- Caffe models

Dependencies: OpenCV2, Caffe
